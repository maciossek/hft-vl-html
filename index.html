HTML

  • 1980, gab Tim den Vorschlag für einen besseren Weg für Forscher Dokumente zu teilen und zu lesen
  • 1990 Vorschlag für das WWW

HTML

  • Hyper Text Markup Language
  • Hypertext: Überwinden der Beschränkungen von geschriebenen Text. Interaktiv.
  • Markup Language: "Aufpeppen" eines Dokuments mit Attributen wie Schriftgrößen, Bildern, Listen, Links zu anderen Seiten…
  • HTML ist in normalen Textdateien geschrieben.

HTTP

  • Hyper Text Transfer Protocol
  • Stateless

HTTP Kommandos

  • GET: Hole Daten vom Server
  • POST: Hole Daten vom Server und sende Daten, z.B. Formulardaten
  • HEAD: Wie GET, aber ohne die eigentlichen Daten des Bodies zu senden (u.a. zum Cache-Check zur Aktualitätsprüfung im Browsertests)
  • PUT: Update von Daten
  • OPTIONS: Anforderung einer Liste vom Server unterstützter Optionen
### HTTP Request Header (mathias-maciossek.com) ``` :authority:mathias-maciossek.com :method:GET :path:/ :scheme:https accept:text/html,application/xhtml+xml,application/xml;q=0.9,image/webp,image/apng,*/*;q=0.8 accept-encoding:gzip, deflate, br accept-language:en,de;q=0.8,fr;q=0.6,en-US;q=0.4 cache-control:no-cache dnt:1 pragma:no-cache upgrade-insecure-requests:1 user-agent:Mozilla/5.0 (Macintosh; Intel Mac OS X 10_12_5) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/59.0.3071.115 Safari/537.36 ```
### HTTP Response Header (mathias-maciossek.com) ``` content-encoding:gzip content-length:659 content-type:text/html date:Sat, 15 Jul 2017 11:59:43 GMT etag:W/"58b7e2b7-570" last-modified:Thu, 02 Mar 2017 09:15:35 GMT server:nginx/1.11.10 status:200 vary:Accept-Encoding ```
### HTTP Zustandcodes ``` 1xx (information) 100 Continue 101 Switching protocols 2xx (Success) 200 ok 201 Created 202 Accepted 204 No Content 3xx (Redirect) 300 Multiple Choices 307 Temporary Redirect ```
### HTTP Zustandcodes ``` 5xx (Server error) 500 Internal Server error 501 Not implemented 4xx (Client error) 400 Bad Request 401 Not Authorized 404 Not found ```
